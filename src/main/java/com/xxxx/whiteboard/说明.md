视图是一个”虚表”，用大白话说，就是从已经存在的表的全部字段或数据中，
挑选出来一部分字段或数据，组成另一张”并不存在的表”,这张虚表被称之”视图”。
视图中的字段与对应的数据均来自已经存在的表。对于视图来说，这些已经存在的表就
被称为”基表”。 基表可以是一张表，也可以是多张表。视图的本质可以理解为一条查询
语句，视图中显示的结果， 就是这条查询语句查询出的结果。
